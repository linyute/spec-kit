# 支援

## 如何提交問題並獲得協助

此專案使用 GitHub 問題來追蹤錯誤和功能要求。請在提交新問題之前搜尋現有問題，以避免重複。對於新問題，請將您的錯誤或功能要求作為新問題提交。

如需有關使用此專案的協助或問題，請：

- 開啟 [GitHub 問題](https://github.com/github/spec-kit/issues/new) 以取得錯誤報告、功能要求或有關規格驅動開發方法的問題
- 查閱 [綜合指南](./spec-driven.md) 以取得有關規格驅動開發流程的詳細文件
- 查閱 [README](./README.md) 以取得入門說明和疑難排解提示

## 專案狀態

**Spec Kit** 正在積極開發中，並由 GitHub 員工**和社群**維護。我們將盡力及時回應支援、功能要求和社群問題。
